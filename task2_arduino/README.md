# Task 2 – Arduino Servo Control (Tinkercad)

## Overview
This project controls a servo motor using a potentiometer.

## Components
- Arduino Uno
- Servo Motor
- Potentiometer
- LED
- Resistor

## Functionality
- Potentiometer controls servo angle
- Servo is limited to 0°–68°
- If angle > 68°, LED turns ON
- Otherwise LED remains OFF

## Example
- 30° → Servo moves, LED OFF
- 70° → Servo stops at 68°, LED ON
